# Installation

1. To install a solution pack, click **Content Hub** > **Discover**.
2. From the list of solution pack that appears, search **FortiRecon Automation Services**.
3. Click the **FortiRecon Automation Services** solution pack card.
4. Click **Install** on the lower part of the screen to begin the installation.

## Prerequisites

The **FortiRecon Automation Services** solution pack is not dependent on any other solution pack.

# Configuration

The **FortiRecon Automation Services** solution pack does not require any configuration.
