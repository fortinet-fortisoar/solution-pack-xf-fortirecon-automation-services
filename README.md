# Overview

The **FortiRecon Automation Services** solution pack provides automated playbooks to streamline incident response for External Attack Surface Management (EASM) and Brand Protection (BP) use cases. It enables security analysts to efficiently manage *Critical* and *High-severity* issues by automating ticket creation in Jira and sending notifications for relevant services.

Key use cases include:
- Handling systems, services, or devices that are public-facing
- Identifying typosquatted domains and phishing threats
- Detecting code repository exposures.

Each playbook follows a consistent workflow—triggering on active issues, retrieving metadata, formatting issue details into HTML, and generating tickets or alerts—enhancing SOC efficiency and ensuring timely, structured incident handling.
